# Laundry Wallah / Laundry Mart Website

## Project Description
Laundry Wallah / Laundry Mart is a simple and user-friendly website created to represent a professional laundry service. The website provides information about the business, including services offered and a clear price list. It is designed to help customers understand available laundry solutions such as washing, ironing, dry cleaning, and doorstep pickup and delivery. This project is suitable for learning basic web development and creating service-based websites.

## Technologies Used
- HTML5


## How to Run the Project
1. Download or clone the project files to your computer.
2. Locate the `index.html` file in the project folder.
3. Simply open `index.html` in any web browser (Google Chrome, Edge, Firefox, etc.).
4. The website will load and display the laundry service information.

## Project Structure
- `index.html` – Main webpage of the Laundry Wallah / Laundry Mart website
