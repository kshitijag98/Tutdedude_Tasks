# Laundry Services - Responsive Web Page

## 📌 Project Overview
This project is a responsive Laundry Services web page built using **HTML** and **CSS**.  
The task focuses on improving the responsiveness of the existing desktop layout so it works properly on **tablet** and **mobile** devices using **CSS Media Queries** and **Flexbox**.

---

## 🎯 Objective
Fix the responsive issues in the Laundry Services Hero Section and Navbar so the layout adapts smoothly across different screen sizes.

---

## 🛠️ Features Implemented

### ✅ Navbar Responsiveness
- Reduced logo and text size for tablet screens.
- Hidden navigation links on mobile view.
- Displayed only the logo and username on smaller devices.
- Replaced `display: inline` with `display: flex` for better alignment.
   and also adding Hover effect to Navbar list

### ✅ Hero Section Responsiveness
- Reduced text and image sizes for tablet devices.
- Changed alignment and positioning of content.
- Used `flex-direction: column` for mobile/tablet layout.
- Improved spacing and readability on smaller screens.

---

## 📱 Responsive Design
The webpage is optimized for:

| Device Type | Screen Size |
|-------------|-------------|
| Desktop     | 1024px and above |
| Tablet      | 768px – 1023px |
| Mobile      | Below 768px |
