This project is designed to help students understand and practice different types of CSS Selectors by creating and styling a simple HTML webpage. Students will learn how to connect CSS with HTML and apply styles using element selectors, class selectors, id selectors, and nested selectors.

The project includes creating headings, paragraphs, and lists with different styles such as text colors, background colors, and font sizes. It also focuses on applying styles without using inline CSS, helping students understand the proper use of external CSS files.

By completing this project, students will improve their skills in:

Creating HTML structure
Linking external CSS files
Using CSS selectors effectively
Styling text and backgrounds
Understanding class and id usage
Applying styles to specific elements

This task builds a strong foundation in web design and front-end development using HTML and CSS.
